Основые роли:

1. Класс посетитель (например там будут переменные по типу кошелька и голод чтобы он не сожрал весь ресторан xD)
2. Класс официант (можно сделать функции для просмотра меню ресторана, и функцию для выбора чего-либо из меню)
3. Класс ресторан (будет хранить в себе данные о столе посетителя, меню, и функции по типу выбора столика)
4. Код в main (там можно сделать менюшку для пользователя, чтобы он мог взаимодействовать с рестораном)
5. Класс бухгалтер (можно там просто сделать функцию, и она будет принимать в параметрах цену и кошелёк посетителя для проверки)
6. Класс бармен (будет содержать все то, чем обычно занимаются бармены)
